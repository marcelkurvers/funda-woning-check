## Tips for Using the Color System

### âœ… Do's
- **Use colors as a quick scan** - Quickly identify areas of concern
- **Read the explanations** - Each colored metric includes a badge explaining why
- **Compare across chapters** - Colors mean the same thing throughout the report
- **Focus on red items first** - These require immediate attention

### âŒ Don'ts
- **Don't rely solely on colors** - Always read the detailed analysis
- **Don't ignore orange items** - They may become red if not addressed
- **Don't assume green means perfect** - It means good relative to standards
- **Don't compare different metrics** - Each has its own thresholds

---

## Accessibility

We've designed the color system to be accessible:
- **High contrast colors** - Easy to distinguish
- **Icon support** - Visual indicators beyond just color
- **Text explanations** - Every color has a text explanation
- **Consistent patterns** - Same colors mean the same thing

If you have difficulty distinguishing colors, focus on:
1. The **explanation badges** below each metric
2. The **text values** themselves
3. The **detailed analysis** in each chapter

---

## Questions?

If you're unsure about what a color means:
1. Look for the **explanation badge** below the metric
2. Check the **color legend** on the Executive Summary
3. Read the **detailed analysis** in the chapter
4. Refer to this guide for specific thresholds

---

**Remember:** Colors are a guide, not a verdict. Always read the full analysis and consult with professionals (makelaars, bouwkundigen, etc.) before making decisions.
